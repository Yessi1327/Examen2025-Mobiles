# Examen2025-Mobiles
Examen Consumiendo una Api

Nombre: Yessica Lora Vázquez
Matrícula: A01273574
Plataforma: Android (Kotlin + Jetpack Compose)

Este proyecto es una aplicación móvil desarrollada como parte del examen final del curso.
La app permite generar, visualizar y resolver Sudoku en dos tamaños (4x4 y 9x9) utilizando la API pública de API Ninjas — Sudoku.

Incluye selección de dificultad, verificación de solución, manejo de errores, y soporte para funcionamiento offline mediante caché local.
